Accessible Footnote Solution: One-to-one Footnote References

Introduction

The purpose of this page is to demonstrate an accessible solution for footnote references that have a one-to-one correlation between the footnote reference in the page content and the footnote itself. This concept can be applied to footnotes that are at the bottom of a page, within a page or in a modal dialog.

Accessibility Features:

  • Programmatic connection between the footnote reference and the footnote: Selecting, via mouse or keyboard, a footnote reference in the content will move focus to the footnote number at the beginning of the footnote. Selecting the footnote number will return focus to the footnote reference in the content.
  • Screen reader context: CSS is used to add additional content to each link for screen reader users. The footnote reference link is prepended with the word "Footnote". The footnote number link is prepended with the phrase "Return to reference".
