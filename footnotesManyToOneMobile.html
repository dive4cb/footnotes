Accessible Footnote Proof of Concept: Mobile Footnote References (Only Test with Mobile Device)

Introduction

The purpose of this page is to enhance the one-to-one and many-to-one accessible footnote soltuions by providing a modal dialog that contains the footnote information for mobile platforms (phone or tablet). This modal overlay has the advantage of not causing the viewport to scroll all the way down a potentially very long page and then back up. This example is a Proof of Concept and is NOT approved for use at State Farm at this time.

Accessibility Features:

  • Programmatic connection between the footnote reference and the footnote: The programmatic connection present in the desktop view is unchanged. However, in a mobile view, a modal overlay is added to display the footnote content instead of moving the focus down to the bottom of the page and then back up. When the modal is closed, focus returns to the footnote reference in the content.
  • Screen reader context: CSS is used to add additional content to each link for screen reader users. The footnote reference link is prepended with the word "Footnote". The footnote number link is prepended with the phrase "Return to reference".
